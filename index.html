🚀 Installation Guide

1

Get Google Cloud

Download Google Cloud app or visit shell.cloud.google.com

2

Setup Cloud Shell

Open Google Cloud app and click the terminal/shell icon

3

Install RDP

Run these commands in your cloud shell:

sudo apt update -y && sudo apt upgrade -y
sudo apt install wget -y
sudo wget https://dl.google.com/linux/direct/chrome-remote-desktop_current_amd64.deb
sudo apt install ./chrome-remote-desktop_current_amd64.deb -y
sudo DEBIAN_FRONTEND=noninteractive apt install --assume-yes xfce4 desktop-base dbus-x11 xscreensaver
sudo bash -c 'echo "exec /etc/X11/Xsession /usr/bin/xfce4-session" > /etc/chrome-remote-desktop-session'
sudo systemctl disable lightdm.service
4

Get Chrome Remote Desktop

Download the Chrome Remote Desktop app

5

Setup Chrome Remote Desktop

Follow these steps to connect to your cloud computer:

1. Open Chrome Remote Desktop app

2. Click "Get Started"

3. Enable Remote Access

4. Set a secure PIN

5. Connect to your computer

6. Access your Linux desktop

6

Enjoy Your Cloud Computer

You now have access to a powerful Linux environment! You can:

  • Install and run Linux applications
  • Develop and test software
  • Run Linux games and tools
  • Access your files from anywhere
